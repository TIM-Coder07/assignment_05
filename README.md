<!-- Ans to The Q. No. 1: -->
var: var is function-scoped and can be redeclared.
let: let is block-scoped and cannot be redeclared in the same scope. const: const is also block-scoped but its value cannot be reassigned after initialization.

<!-- Ans to The Q. No. 2:  -->
The spread operator (...) allows an iterable (like an array or object) to be expanded in places where zero or more elements or properties are expected.

<!-- Ans to The Q. No. 3:  -->
map: map() creates a new array by transforming each element.
filter: filter() creates a new array with elements that pass a condition.
forEach: forEach() executes a function on each element but does not return a new array.

<!-- Ans to The Q. No. 4: -->
An arrow function is a concise syntax for writing functions using =>, often providing lexical this binding.

<!-- Ans to The Q. No. 5: -->
Template literals are string literals enclosed by backticks ` that allow embedded expressions with ${expression} and multi-line strings.
